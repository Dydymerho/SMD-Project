feat: Implement CLO-PLO Mapping API

## Summary
Triển khai đầy đủ REST API cho quản lý CLO-PLO Mapping với 12 endpoints chính và 2 enhanced endpoints.

## New Files
- CLOPLOMappingResponse.java: DTO for mapping responses
- CLOPLOMappingRepository.java: Repository with custom queries
- CLOPLOMappingService.java: Business logic with validation
- CLOPLOMappingController.java: REST API controller

## Updated Files
- CLOResponse.java: Added ploMappings field
- PLOResponse.java: Added totalMappedCLOs and cloMappings fields
- CLOService.java: Added getCLOWithMappings() method
- PLOService.java: Added getPLOWithCoverage() method
- CLOController.java: Added /with-mappings endpoint
- PLOController.java: Added /with-coverage endpoint

## Documentation
- docs/CLO_PLO_MAPPING_API.md: Complete API documentation
- core-service/CLO_PLO_MAPPING_README.md: Implementation guide
- scripts/test-clo-plo-mapping.ps1: Test script
- IMPLEMENTATION_SUMMARY.md: Project summary

## Features
- ✅ CRUD operations for CLO-PLO mappings
- ✅ Batch create mappings
- ✅ Query by CLO, PLO, Syllabus, Program
- ✅ Enhanced CLO/PLO responses with mapping info
- ✅ Validation: duplicate check, mapping level validation
- ✅ Comprehensive logging
- ✅ Error handling with clear messages
